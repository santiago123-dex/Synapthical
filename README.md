# Synapthical Sport Landing

Landing page desarrollada con Astro + Tailwind para Synapthical Sport.

## Desarrollo

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Edición rápida de contenido

Todos los textos, colores, navegación, servicios y datos de contacto están centralizados en:

- `src/data/site.ts`
